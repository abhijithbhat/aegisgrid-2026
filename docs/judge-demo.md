# Seven-minute judge demo

1. **Orient (45 seconds).** Open the command center, name the single supervisor persona, point to the live event phase and readiness score, then select the top-priority incident from the real heap.
2. **Explain (75 seconds).** In Incident Intelligence, compare the deterministic risk score with AI severity. Expand supporting source IDs, contradiction, missing information, and the approval gate. State that concise evidence—not private chain-of-thought—is shown.
3. **Inject unfamiliar data (2 minutes).** Open Data Injection Lab, upload a CSV with unfamiliar headings, review the editable preview, inspect proposed mappings and confidence, correct one low-confidence zone field, approve, then download the validation report. Reset.
4. **Prove GenAI essentiality (75 seconds).** Run Conflicting Smoke Reports. Show two smoke reports, a theatrical-fog claim, and normal air quality remaining contradictory. Toggle or demonstrate degraded mode: deterministic risk and route remain; semantic synthesis and announcement disappear honestly.
5. **Prove deterministic routing (60 seconds).** Run Accessible Corridor Blockage. Require an accessible route and show the graph engine avoiding stairs/blocked edges, primary and alternate ETA, and comparison with naive distance.
6. **Human control (45 seconds).** Modify one step, assign a team, approve it, and show the append-only audit entry. Emphasize that AegisGrid never dispatches autonomously.
7. **Close (20 seconds).** “AegisGrid turns fragmented signals into one safe, explainable decision—while keeping the trained supervisor accountable.”

Use seed `2026` for a reproducible timeline. Keep network tools open beforehand; the app remains demonstrable if Gemini is temporarily unavailable.
